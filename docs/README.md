# Google Programming Contest — Interactive Storyteller

This folder holds contest materials for the **Google Cloud Next 2026**–style program (see [awards.txt](awards.txt)).

## Files

| File | Purpose |
|------|---------|
| **awards.txt** | Prize categories and eligibility (Live Agent, Creative Storyteller, UI Navigator, subcategories, Grand Prize). |
| **CONTEST_STRATEGY.md** | How the Storyteller codebase maps to each category, recommended primary/secondary entry, proof points, and what to create. |
| **submission_storyteller.md** | Ready-to-edit draft for the **Live Agent** category submission form (description, tech list, demo link placeholders). |
| **submission_live_agent.md** | Legacy draft for the ADAI project (reference only). |
| **IDEAS.md** | Brainstorm of what to do: demo ideas, quick tweaks, feature ideas. |
| **WOW_IDEAS.md** | Real “wow” ideas: director’s commentary mode, “what would Amelia say here?”, pipeline status. |

## Recommended entry

1. **Primary:** **Live Agent** — You use both ADK (agent orchestration) and Gemini Live API (real-time voice story making), all on Google Cloud.
2. **Secondary:** **Creative Storyteller** — Same stack; emphasize interleaved text + audio narrative content.

See **CONTEST_STRATEGY.md** for the full plan and **submission_live_agent.md** for copy-paste submission text.
